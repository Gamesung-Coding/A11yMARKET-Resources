# 최종 프로젝트 기획안
## 프로젝트 팀명
---
**겜성코딩**

## 프로젝트 팀원
---
- **팀장:** 안규태
- **팀원:** 김수민, 백여랑, 신운용, 장지훈

## 프로젝트 주제
---
사용자 맞춤형 접근성 프로필을 기반으로 하는 상품 중계 쇼핑몰

## 프로젝트 수행 방향
---
본 프로젝트는 정보 접근에 어려움을 겪는 사용자와 판매자를 연결하는 ‘상품 중개 쇼핑몰’을 구현한다.

이를 위해, 판매자가 상품을 등록하고 관리할 수 있는 판매자 기능과, 구매자가 상품을 탐색하고 구매할 수 있는 핵심 커머스 기능을 구축한다

## 프로젝트 목적
---
본 프로젝트의 목적은 접근성에 특화된 **개인 맞춤형 UI/UX 설정**과 **판매자의 신뢰도 분석**을 통해 신체적 · 감각적 제약으로 인해 정보 접근이 어려운 계층의 디지털 격차를 해소하는 포용적인 온라인 쇼핑 환경을 구축하는 것이다.

이를 위해, 사용자의 신체적 · 감각적 특성과 필요에 맞춰 ▲고대비 모드, ▲큰 글씨 설정 등을 **'접근성 프로필'** 형태로 직접 선택하고 저장하여 UI 편의성을 극대화한다. 특히, 이번 프로젝트에서는 노약자 등 인지적인 제약을 가진 사용자와 시각적 제약을 가진 사용자를 상정하여 편의성 기능을 구현한다.

또한, ▲ AI를 활용하여 복잡한 상품정보 요약 설명 생성, ▲ TTS(음성 변환) 서비스를 제공하여, 시각적 · 인지적 제약이 있는 사용자도 쉽게 상품 정보를 명확히 인지하고 구매 절차를 진행할 수 있도록 돕는다.

나아가, '상품 중개' 플랫폼의 핵심인 신뢰성 확보를 위해 ▲판매자 인증 시스템과 ▲신뢰가능한 결제 대행 서비스, ▲구매 확정 시스템을 도입한다. 이를 통해 모든 사용자가 정보 접근의 제약 없이, 신뢰할 수 있는 판매자로부터 필요한 상품을 편리하게 비교하고 구매할 수 있는 플랫폼의 구현을 최종적인 목표로 한다.

## 필수 기능
---
본 프로젝트의 기능 목록은 “**MVP(최소 기능 제품) 구현**” **과** “**고도화** **목표**” 로 나누어 지며, 최소 기능 제품 기능 목록은 본 개발 기간 내 구현하게 될 목록이며, 고도화 목표는 추후 확장성 고려 항목이자, 추가적인 개발 시 포함될 기능목록이다.

**최소 기능 제품 기능 목록**
1. **공통 및 회원기능**
    - **회원 관리**: 회원가입, 로그인, 로그아웃, 회원탈퇴
    - **마이페이지**: 회원정보 수정, 주문 내역조회, 접근성 프로필 설정
2. **구매자 기능**
    - 상품 목록 조회 (검색/정렬/필터링)
    - 상품 상세 조회
    - 장바구니 관리
    - 상품 결제
    - 주문 취소 요청
3. **판매자 기능**
	- 판매자 가입 신청 (정보 입력, 사업자 등록증 업로드)
	- 상품 등록 (관리자 승인 후 노출)
	- 주문 접수 및 배송 처리
	- 판매 현황 대시보드
4. **플랫폼 신뢰 시스템**
	- 신뢰성 있는 결제 대행 서비스 이용 (Toss 등)
	- 구매확정 시스템
	- 구매 확정 후 판매자 정산 시스템
    - 판매자 등급제 시스템 (신규 / 인증/ 우수)
5. **관리자 기능**
    - 판매자 가입 및 인증 승인/반려
    - 판매자 상품 승인/반려
    - 회원 목록 조회, 권한 관리
    - 주문 내역 및 상태 관리
6. **접근성**
    - 접근성 프로필을 통한 맞춤형 UI 환경 지원 기능
	- AI 기반 상품 상세 설명 요약 기능
	- 모든 텍스트 정보에 대한 기기 TTS 지원
    - 접근성 인증 판매자 마크 부여 시스템

**향후 확장성/고도화 목표**
- **탐색 경험 향상**
	- 접근성 요구 기반의 상품 추천
	- 상세한 리뷰 우선 노출
- **접근성 고도화**
	- AI 음성 지원 (Gemini + Web Speech API)
	- WCAG 2.1 준수
	- 접근성 기능을 통한 탐색, 리뷰 게시 등
- **신뢰 시스템 고도화**
	- 후기(리뷰) 시스템 도입
	- 판매자 자동인증 (사업자 진위 확인 API 연동 등)
	- 판매자 신분증 인증
- **AI 기능**
	- 판매자를 위한 상품 '대체 텍스트' 초안 생성
	- 판매자를 위한 상품 정보 입력 가이드
- **결제 고도화**
	- 정기 주문 결제
